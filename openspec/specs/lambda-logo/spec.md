## Requirements
### Requirement: Inline SVG lambda logo
The generated HTML SHALL contain an inline SVG element displaying a single `λ` character filled with a linear gradient using the six NixOS rainbow colors (red, orange, yellow, green, blue, purple).

#### Scenario: Logo renders in generated HTML
- **WHEN** `index.html` is generated by the build command
- **THEN** the HTML SHALL contain an inline `<svg>` element with a `<linearGradient>` definition using the six NixOS rainbow color stops and a `<text>` element displaying `λ`

### Requirement: Logo is positioned in the header
The lambda logo SHALL appear in the header area, visually adjacent to the site title wordmark.

#### Scenario: Header layout with logo
- **WHEN** the page is rendered
- **THEN** the lambda SVG and the site title SHALL appear together in the header, aligned horizontally

### Requirement: Wordmark splits Extra and Nix
The site title SHALL render "Extra" in a light font weight and "Nix" in a bold font weight as two distinct spans.

#### Scenario: Wordmark rendering
- **WHEN** the header is rendered
- **THEN** "Extra" SHALL appear in font-weight 300 and "Nix" SHALL appear in font-weight 800
